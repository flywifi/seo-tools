# Adaptation Engine

Every output is adapted along five axes before it is finalized. State the assumptions chosen when
the user has not specified them.

## Axis 1: Skill level
- Beginner: more steps, each one action; name tools and techniques plainly; add "watch out for" notes
  at the spots beginners struggle; normalize fixes for mistakes.
- Intermediate: tighter steps, fewer basics, room for technique variation and upgrades.
Default to beginner-friendly with intermediate options noted, since the audience skews beginner to
intermediate.

## Axis 2: Tenure (renter vs owner)
- Renter-friendly: reversible, damage-free, landlord-safe methods (tension rods, peel-and-stick,
  freestanding, removable adhesives, no permanent paint or holes). Always offer this version for any
  project that could be permanent, because The Renter is a core persona.
- Owner / permanent: the full-commitment method (paint, paneling, hardware, built-ins).
Where a project has both, present the renter-friendly path and the permanent path side by side.

## Axis 3: Budget tier
- Budget: lowest-cost materials and a thrift or Marketplace sourcing option.
- Mid-range: the default; balanced cost and durability.
- Premium: the splurge version where it earns its place.
Give material and tool lists with budget vs premium alternatives, and flag where a thrifted find can
replace a new purchase. Never invent specific prices; give honest ranges and say when a price
depends on local sourcing.

## Axis 4: Persona target
Map the output to one or two named personas (The Renter, The Vintage Hunter, The Organizer, The
Holiday Maximalist, The New Homeowner) and tailor the framing, examples, and pain point addressed.
State which personas a piece serves.

## Axis 5: Surface (platform)
Adapt format and length to the destination using platform-engine: a long-form build for YouTube, the
extracted moments as Shorts, Reels, and TikToks (15 to 60 seconds, audio-first, vertical), a 2:3
keyword-rich Pin for Pinterest, and a 4:5 or 3:4 carousel for an Instagram step-by-step. Honor the
content ecosystem ratio from brand-engine: one long-form piece, three to five short-form pieces, and
one to three pins or carousels.

## Adaptation order
1. Identify skill level, tenure, budget, and persona (ask or assume and state it).
2. Generate the core project or content in Alex's voice and aesthetic.
3. Produce the renter-friendly and budget alternatives where relevant.
4. Adapt to each target surface per platform-engine.
5. Hand to quality-review against the rubric.
