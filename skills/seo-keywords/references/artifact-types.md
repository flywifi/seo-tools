# seo-keywords artifact types

## SEO strategy package
A complete keyword and competitive strategy for one topic. Required elements: keyword cluster (primary, secondary, long-tail tiers), volume estimates labeled "[estimated, unverified]" with a verification note, search intent classification with format implication, competitive gap analysis per platform (overserved/underserved angles, gap summary, confidence), title skeleton (2 to 4 options, keyword front-loaded, character counts), description skeleton (per platform), retrieval gaps list, and a govern-artifact gate result.

## Keyword cluster
A tiered keyword set for one topic and platform. Required elements: 1 to 2 primary keywords, 2 to 4 secondary keywords, 4 to 8 long-tail keywords, a difficulty note for high-competition primaries, and volume estimate ranges labeled "[estimated, unverified]."

## Competitive gap report
Overserved and underserved angles for one topic and platform. Required elements: list of overserved angles with evidence source, list of underserved angles with differentiation opportunity, gap summary sentence, and confidence level (high/medium/low) reflecting retrieval quality.
