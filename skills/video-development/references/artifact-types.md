# video-development artifact types

## Production package
The full build of one concept. Required elements: a hook, title options with character counts, the
six outline sections, a thumbnail concept, an SEO-aware description, and 3 or more standalone
short-form clips. Title and thumbnail must align; the SEO description carries 1 to 2 primary keywords
in the opening 200 characters.

## Short-form clip set
The 3 or more clips extracted from the concept. Required elements: each clip has its own first-3-
seconds hook, a format, and a target length, and works with no prior context from the full video.
