# shortform-repurposing artifact types

## Short-form content package
A complete repurposed content set from one long-form project. Required elements: 3 to 5 clip proposals (each with hook, transcript excerpt or conceptual summary, and platform tag), per-platform captions (each within character limits and with FTC disclosure line if required), per-platform hashtag sets (tiered: broad, niche, micro), 1 to 3 Pinterest Pins, a staggered drop schedule from calendar-slot, and a govern-artifact gate result.

## Clip extract
One standalone short-form clip proposal. Required elements: clip number, hook (3 to 5 words opening), content summary or transcript excerpt, platform recommendation, duration estimate, and a note confirming the clip works as a standalone without context from the long-form.

## Platform caption set
Captions for one clip across multiple platforms. Required elements: one caption per platform, each within platform character limits, with hook line, body, CTA, FTC disclosure line (if required), and character count.
