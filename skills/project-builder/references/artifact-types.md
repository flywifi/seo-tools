# project-builder artifact types

## Project brief
A structured DIY project plan. Required elements: snapshot (title, pillar, persona, transformation arc, difficulty, estimated hours, renter_friendly flag, materials preview), materials list (grouped by category, cost ranges labeled as estimates, safety-required items flagged), numbered step sequence (each step with action, detail, safety note, broll tag, and licensed_trade_required flag), 2 to 3 styling variants, renter alternatives for relevant steps, and a govern-artifact gate result.

## Project snapshot
A one-page project overview. Required elements: title, pillar, persona served, transformation arc (before/process/after), difficulty, estimated hours, renter_friendly, budget tier, and a 3 to 5 item materials preview.

## Safety report
A flagged list of any steps or materials in a project that require a licensed trade (electrical, gas, structural, load-bearing). Required elements: each flagged step with licensed_trade_required: true, the specific trade type, and the note "licensed professional required; this step is out of scope for DIY."
